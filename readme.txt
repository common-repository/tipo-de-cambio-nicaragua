== Description ==

El **Tipo de Cambio Nicaragua** es un Widget que brindar información de hoy, ayer, mañana y promedio del cambio del dólar (USD) a córdoba nicaragüense. Es una herramienta útil para PYMES y tener de primera mano toda la información necesaria en el sitio web de quien la instale. El Tipo de Cambio de Nicaragua es un Widget configurable por el usuario.

Toda la información que muestra este complemento es de dominio público, brindada por el Banco Central de Nicaragua en el [sitio web](https://www.bcn.gob.ni/servicio-web-tipo-de-cambio) el cuál cita textualmente *Los Servicios Web están a su disposición de forma gratuita, para facilitar la consulta del tipo de cambio.*

=== Características ===

1. Permite conocer el tipo de cambio del día
2. Muestra el tipo de cambio de ayer y mañana
3. Muestra el promedio del tipo de cambio, que es la suma de todos valores entre la cantidad de días

== Instalación ==

1. Visite la página de plugins dentro de su escritorio de Wordpress y seleccione 'Añadir nuevo'
2. Busque **Tipo de Cambio Nicaragua**
3. De clic en instalar ahora
4. Active el Tipo de Cambio Nicaragua desde su página de plugins
5. ¡Terminaste!

== Configuración ==

1. Visite la página de Widgets
2. De clic en botón parte superior izquierda y busque en la sección de widget **Tipo de Cambio Nicaragua**
3. De clic en el Widget
4. El título por defecto se le agregará un titulo que es "TIPO DE CAMBIO NICARGUA" pero podrá ser cambiado por el usuario.
5. Un subtitulo que describe el tipo de conversión que muestra Ej. Cambio de Dólar (USD) a Córdoba nicaragüense (NIO)
6. El usuario puede mostrar o quitar el logo
7. El usuario puede mostrar o quitar el detalle
